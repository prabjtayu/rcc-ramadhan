# **🌙 Ramadhan Portal \- Teman Ibadah Digitalmu\!**

Halo\! 👋 Selamat datang di *repository* Ramadhan Portal.

Project ini aku bangun khusus sebagai pusat layanan ibadah digital untuk menemani keseharian di bulan suci Ramadhan. Nggak cuma tampilannya yang dibikin *clean* dan *eye-catching*, tapi web ini juga dirancang supaya gampang banget diakses lewat HP, tablet, maupun laptop kamu. Semua ada dalam satu layar\! ✨

## **🔥 Ada Fitur Apa Aja Sih?**

Daripada pusing buka banyak aplikasi, di web ini aku udah kumpulin 5 fitur krusial yang paling dibutuhin selama Ramadhan:

* ⏱️ Jadwal Imsakiyah Anti-Telat: Terkoneksi langsung dengan API MyQuran. Kamu tinggal pilih kota, dan *boom\!* jadwal shalat serta imsak sebulan penuh langsung muncul. Tanggal hari ini juga bakal otomatis ke-highlight lho\!  
* 🤲 Kumpulan Doa Pilihan: Biar puasanya makin afdol, ada contekan niat puasa, doa berbuka sesuai sunnah, dan doa-doa harian lainnya.  
* 📿 Tasbih Digital (Dzikir Tracker): Suka lupa udah dzikir berapa kali? Pake fitur ini aja. Ada target hariannya juga biar makin semangat.  
* 💰 Kalkulator Zakat Pintar: Nggak usah pusing ngitung manual. Masukin aja gajimu atau jumlah emas yang kamu punya, sistem bakal otomatis ngecek apa kamu udah wajib zakat (mencapai nisab) dan ngitungin nominalnya.  
* 📋 Mutabaah Yaumiyah (To-Do List): Tracker ibadah super lengkap\! Mulai dari checklist shalat 5 waktu, target bacaan Al-Qur'an, sampai kalender visual puasa 30 hari ada di sini.

## **🛠️ Dibangun Pake Apa Aja?**

Project ini sengaja aku bikin seringan mungkin tanpa perlu instalasi server atau *framework* yang berat. *Tech stack* yang aku pakai:

* HTML5: Bikin struktur web yang rapi dan sesuai standar.  
* Tailwind CSS (via CDN): Rahasia di balik desain cantiknya\! Bikin styling UI/UX jadi gampang banget dan dijamin *fully responsive*.  
* Vanilla JavaScript: Otak di balik semua interaksinya. Mulai dari perpindahan halaman ala *Single Page Application (SPA)*, rumus kalkulator, sampai sistem checklist datanya.  
* Fetch API: Buat narik data jadwal shalat secara dinamis (langsung dari internet).

## **🚀 Cara Cobain Langsung di Laptop Kamu**

Gampang banget kok, nggak butuh *install* macem-macem:

1. Unduh file project ini (klik Code \> Download ZIP) lalu *extract* foldernya.  
2. Cari file yang namanya index.html.  
3. Langsung *double-click* aja file itu\! Otomatis bakal kebuka di *browser* kesayanganmu (Chrome, Safari, dll).  
4. Penting: Pastiin laptop kamu lagi konek ke internet ya, biar *style* Tailwind dan data jadwal Imsakiyah-nya bisa ke- *load* dengan sempurna.

## **🌍 Cara Bikin Web Kamu Online (Gratis\!)**

Biar hasil karyamu ini bisa dipamerin dan dipake sama temen-temen atau keluarga, kamu bisa *hosting* web ini secara gratis. Ini 2 cara paling gampang:

### Pilihan 1: Pake Netlify (Sumpah, ini gampang banget)

1. Buka situs [Netlify Drop](https://app.netlify.com/drop).  
2. Tinggal seret (*drag and drop*) folder project kamu ke lingkaran putus-putus di layar.  
3. Tunggu beberapa detik, dan *voila\!* Netlify bakal ngasih kamu *link* URL yang udah *live* dan siap disebar.

### Pilihan 2: Pake GitHub Pages (Lebih terlihat Pro\!)

1. *Upload* file index.html dan README.md ini ke *repository* GitHub kamu.  
2. Masuk ke tab Settings di *repository* tersebut.  
3. Pilih menu Pages di sebelah kiri.  
4. Di bagian *Branch*, ganti dari None ke main (atau master), lalu klik Save.  
5. Tunggu 1-2 menit, coba *refresh* halamannya. Link website kamu bakal muncul di bagian paling atas\!
